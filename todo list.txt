
=====================================================================
speed changes
=====================================================================
Some small customizations for speed are needed. 
Before starting to work, i suggest reading all the rest of this section because it is all related.
First you need to know some quircks of this system:
    > you currently start out so slow because the previous framework made you start on first ring touch, not on moving

    > your speed variable on start or reset currently starts at 100 and it sets you to 100% speed total
    > on touching a ring it adds 3, but then sets you to speed var * 2
    > the above means that after touching first cp, you go from 100% to 206%

    > getting more speed doesnt apply to dash at the same rate as walking would so the multiplier isnt entirely accurate. 
    > we give 6% speed per ring, but this does not translate to acutally having 6% more speed, it works more like 3%
    > the hud i made said u get 3% per checkpoint, wich is not technicually true, but does work about right if you consider the above point 

    > before you reach 168 speed, holding W and Boost works differently. Before this speed holding W does nothing but afterwards you have to hold W during boost to get maximum speed.


- It would be best to start at 200% speed instead of 100. the slow buildup doesnt feel good for players.
- instead of doubeling the speed during set-speed, you can just start with variable 200 and add 6 instead of 3. this would be easier to understand and work with later.

the following 3 customization changes would need to be added to make map making a lot better.
you can make it part of level data, but that would probly require an ingame interface.
diferent maps might need diferent speed, but ingame interface is proboly gona feel clunky. there is no right option here so pick what you want to do more.
if you do make it via map data, please also move ring size customization there (and remember old map data works diferent so it has to handle that)       
    1) cusomtization maximum speed bonus: to prevent long maps from being unplayably fast, a customization to set a maximum speed bonus value
    2) customization base speed: change the speed at which the player starts at, defaulting to 200
    3) cusomtization speed gain per cp: a customization config to set the amound of speed increase you receive per ring touched.

an example of ingame system would be: when u enter mode, up/down to select setting, left/rikght click to adjust setting, v to flip a value or reset to default.
